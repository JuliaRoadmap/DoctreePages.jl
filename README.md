## 关于
这是一个轻量级交互式文档生成器
* 轻量级：只基于 CommonMark 和 TOML
* 交互式：包括页面讨论功能，填空题与条件激发文本
* 简单：即使没有编程经验也能使用
* 强大：[各式各样的功能](docs/zh/features.md)
* 可配置：超过30个设置项（不包括子项）

## 用途
此包**不是专为包文档设计**，更适合用于教育性文档/博客等
